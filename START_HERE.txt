🎯 FUNÇÕES PRINCIPAIS
═════════════════════════════════════════════════════════════════════════════

  ⭐ executeWorldCup(teamsData)
     → Simula TUDO (FUNÇÃO PRINCIPAL)
     → Retorna relatório completo com campeão

  🎪 getQualifiedTeams(teamsData)
     → Lista os 6 campeões continentais
     → Rápido e direto

  📅 getWorldCupCalendar(teamsData)
     → Retorna calendário em dezembro
     → 5 datas de jogos

  🏆 compareQualifiedStrength(qualified)
     → Ranking de força dos 6 qualificados
     → Análise tática

  📈 generateWorldCupStats(report)
     → Estatísticas detalhadas
     → Campeão, força média, etc.

  💾 saveWorldCupResult(year, report)
     → Salva em localStorage
     → Persiste entre sessões

  📂 loadWorldCupResult(year)
     → Carrega resultado anterior
     → Restaura histórico

  📚 listAllWorldCups()
     → Lista todos os Mundiais
     → Histórico completo
