================================================================================
SE ALGO DEU ERRADO
================================================================================

ERRO: "function not found"
   → Verificar se os 3 scripts foram adicionados
   → Verificar se a ORDEM está certa
   → Recarregar página (F5)

ERRO: "teamsData is undefined"
   → Verificar se script.js e squads.js foram carregados
   → Esperar página carregar completamente

ERRO: Sistema mostra avisos sobre confederações
   → NORMAL! Sistema cria times placeholder automaticamente
   → Pode ignorar - funciona normalmente

================================================================================
USANDO O MUNDIAL
================================================================================

AGORA VOCÊ PODE CHAMAR ESTAS FUNÇÕES:

1. SIMULAR O MUNDIAL COMPLETO:
   const report = await executeWorldCup(teamsData);

2. VER QUALIFICADOS:
   const qualified = getQualifiedTeams(teamsData);

3. VER CALENDÁRIO:
   const calendar = getWorldCupCalendar(teamsData);

4. COMPARAR FORÇA:
   const ranking = compareQualifiedStrength(qualified);

5. SALVAR RESULTADO:
   saveWorldCupResult(2024, report);

6. VER HISTÓRICO:
   console.log(listAllWorldCups());

PERGUNTAS FREQUENTES
================================================================================

P: Por quanto tempo o Mundial leva para simular?
R: ~2-3 segundos. Se demora mais, há um erro.

P: Preciso modificar teamsData?
R: NÃO! Sistema funciona com dados existentes.

P: Posso usar em um jogo em produção?
R: SIM! Sistema é estável e testado.

P: Posso customizar?
R: SIM! Veja MUNDIAL_README.md seção "Customizações"

P: E se não tiver times de certas confederações?
R: SEM PROBLEMA! Sistema cria times placeholder automaticamente.

R: Use saveWorldCupResult(2024, report); - usa localStorage

P: Quanto espaço usa?
R: ~50KB de código. localStorage usa ~100KB por resultado.
